# DeKUT SDG Talent Bridge

Institutional innovation and talent management platform for Dedan Kimathi University of Technology.

## Structure
```
backend/   — FastAPI + PostgreSQL
frontend/  — React 18 + TypeScript + Vite
```

## Backend
- FastAPI 0.111.0
- PostgreSQL 15+
- SQLAlchemy 2.0
- JWT Authentication
- Rate limiting + XSS sanitization

## Frontend
- React 18 + TypeScript
- Vite 5
- React Router DOM v6
- Axios

## Version
v2.0.0 — March 2026
